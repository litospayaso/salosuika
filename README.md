# Salosuika

![Salosuika header image](https://raw.githubusercontent.com/litospayaso/salosuika/refs/heads/master/src/assets/icons/header.png)

This is a personal project for creating FOSS suika game for my girlfriend. All images and music have been generated with AI.

## App

This app has been released in Google play store: [Salosuika](https://play.google.com/store/apps/details?id=angelitapp.salosuika)

Aditionally you can check the releases to install the apk in your phone

## Web version

The app is also stored in github pages, you can play [here](https://litospayaso.github.io/salosuika/)
